### 개발 서비스
- 머신러닝을 활용한 선물 추천 기능
- 내가 받고 싶은 선물 등록 및 친구가 받고 싶은 선물 확인 기능
- 개발 완료 후 추가적으로 PWA 기술 적용

### 데이터 수집
- 네이버 쇼핑 API

### 기술
- front-end : Vue (SPA web application)
- serverless
- Database : firebase
