### 개발 서비스
- 학부, 학과별 홈페이지의 모든 공지사항 하나의 텝에 모아 놓은 공시사항 기능
- 게시판 기능 (수업평가, 익명게시판, 중고거래, 분실물 신고)
- 시간표 기능
- 캘린더 기능

### 기술
- android (java)
- Database : firebase
